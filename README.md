Инструкция:
Установка необходимых библиотек: 
1. pip install -r requirements.txt
2. либо просто
  pip install pytest,
  pip install selenium

Запуск:
pytest main.py
Опциональные параметры 
--browser
(Возможные значения chrome, firefox)
--browser_version
