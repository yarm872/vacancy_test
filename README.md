Инструкция:
Установка необходимых библиотек: 
pip install -r requirements.txt
либо просто 
pip install pytest
pip install selenium

Запуск:
pytest main.py
Опциональные параметры 
--browser
(Возможные значения chrome, firefox)
--browser_version
